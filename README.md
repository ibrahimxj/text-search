# Text Search

Simple text search app built with React, TypeScript, and Vite.

## How to run this

First, you'll need Node.js installed. If you don't have it, grab it from [nodejs.org](https://nodejs.org/).

# Clone and install
git clone <repository-url>
cd text-search
npm install

# Start dev server
npm run dev

That's it! Open http://localhost:5173 and you're good to go.
